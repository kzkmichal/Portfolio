## Getting Started

### Installation

1. Clone the repo

```sh
git clone https://github.com/kzkmichal/Portfolio
```

2. Install NPM packages

```sh
npm install
```

### Available scripts

| Command      | Description            |
| ------------ | ---------------------- |
| `gulp dev`   | Open local server      |
| `gulp build` | Create optimized build |
